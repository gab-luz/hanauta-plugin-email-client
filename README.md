# hanauta-plugin-email-client

Hanauta plugin repository for email client.

## Entrypoints
- hanauta_plugin.py (plugin metadata/registration when present)

## Usage
Install through Hanauta Marketplace or clone into your Hanauta plugins directory.

## Development
This plugin uses `uv` for dependency locking:

```bash
uv sync
python3 email_client.py
```
